## Setup

1. Clone the repository
2. Copy `config.example.js` to `config.js`
3. Add your EmailJS credentials to `config.js`:
   - Get your Public Key from [EmailJS](https://www.emailjs.com/)
   - Create a service and get the Service ID
   - Create a template and get the Template ID
4. Add your images:
   - `swagnik pfp.jpeg` - Profile picture
   - `portfolio bg.png` - Background image
   - `resume.pdf` - Your resume file
5. Open `index.html` in a browser
